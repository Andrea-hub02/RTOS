Descripción
F413HZT6_CTR busca desarrollar un dispositivo que permita controlar la temperatura de un cuarto con la finalidad de regular la condición de un cuarto, de modo, que 
si la habitación esta caliente, se buscará accionar un actuador que se encarge de reducir la temperatura, en cambio, si la temperatura es muy fría, el dispositivo 
se asegurará de apagar el dispositivo hasta que regrese a una temperatura ambiente.
